# API Real Madrid

API desenvolvida para fornecer informações relacionadas ao clube Real Madrid, utilizando rotas para consulta de dados e integração entre backend e frontend.

---

## 🚀 Como executar

### 1. Clonar o repositório

```bash
git clone https://github.com/seuusuario/api-real-madrid.git
```

### 2. Entrar na pasta do projeto

```bash
cd api-real-madrid
```

### 3. Instalar as dependências

```bash
npm install
```

### 4. Executar o projeto

```bash
npm run dev
```

### 5. Abrir no navegador

```bash
http://localhost:3000
```

---

## 📌 Endpoints / Rotas

| Método | URL | Descrição |
|--------|------|------------|
| GET | / | Página inicial da API |
| GET | /players | Lista jogadores |
| GET | /titles | Lista títulos |
| GET | /matches | Lista partidas |
